Web Frontend Development II

WDD 330

Instructor Preparation

Overview

The team activity for this week introduces front end authentication with JWT (Json Web Tokens). In order to do this we need a backend server to generate the token for us. The first part of the activity involves the students pulling down some Node boilerplate that will provide this. If the students completed Front End workflow assignments from the Week 9 readings, then getting this server setup shuold not be a problem. If they did not complete that then it will be somewhat confusing.

Remind students that they should be working on their personal challenges. The only assignment this week is the team activity (~1hr) so they should plan on spending the rest of their time working on their projects. (~7-8 hrs).

Meeting Time

This week you could use your meeting for questions on projects. You should also spend a few minutes talking making sure the server part of the team activity makes sense to the students...one of the biggest challenges is working with multiple servers...their server will be listening on port 3000 and the frontend code will be on a different port. Navigating this could cause confusion.
