Ponder: Team Activity: Calculator

Overview

This activity will pull several of the topics we have recently discussed together. Objects, modules, organization, along with events and DOM manipulation with Javascript. The activity will be to build a calculator.

Instructions

Complete the following assignment as a team. Designate one team member as the "main driver" and collaborate on their copy of the code. Everyone on the team should be actively engaged in writing the code and contributing to the solution. Once the solution is working, make sure that everyone on the team gets a copy of the code. Each week let someone else be the "main driver" of the coding.

Core Requirements

  1. Think through the steps that need to take place for a calculator to function. Write them out. Be as specific in your can...but write them in plain language. My list started like this for example:

    1. wait for a button to be clicked.
    2. check which button it was
      1. If it was a number...
      2. If it was an operation...

    Think about how you would convert your list of steps into code. How could you use modules, objects, or classes to better organize this code? What modules would you need? What objects would make sense? Discuss this with your group.

  2. Create a folder for this project on your computer. Download this starter html and css. Create a javascript file called app5.js, and add a script element to your HTML. We will be using modules, so don't forget to add type="module".

    Create one more JS file called calc.js. Implement your calculator based off of your work in step one as a module in this file.

  3. Style it.

    Add the CSS required to make your list of hikes match this wireframe.

Stretch Goals

  1. Get the details

    When we touch a hike it would be good if it showed us the full details for that hike. Make it so. Replace the list of hikes with a detailed view of one hike.

  2. Getting Back to our list

    Add a "back" button/link that will take you from the hike details back to the main list.
