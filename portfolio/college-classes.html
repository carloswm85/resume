WDD331 - Advanced CSS

Technology Focus: HTML/CSS/SASS

Instructor: Preston S. Blaylock

Deeper learning into topics in cascading style sheets (CSS). Topics include: complex CSS selectors, advanced CSS layout and positioning techniques, CSS transitions and animations, CSS Preprocessing, an introduction to CSS libraries, and using scalable vector graphics (SVG) with HTML and CSS.
WDD330 - Web Frontend Development 2

Technology Focus: Web app using HTML, CSS and JS. AJAX, HTML5 APIs.

Instructor: Andrew Chandler

Note: Taking on Fall 2021

This course is designed to give students the skills required to create web applications using HTML, CSS , and Javascript. It is intended to help the student learn to do this without the aid of third-party frameworks or libraries. Because of this, the course focuses on how to solve larger, ill-structured business problems by designing and creating web applications to be run on mobile devices.

Instructor: Blaine Robertson

CIT111 - Introduction to Databases

Technology Focus: Database Management Systems (DBMS), Relational Databases, SQL, Computer Aided Software Engineering (CASE) with MySQL Workbench.

Note: Taking on Fall 2021

Instructor: John Marlowe

This course covers the basic elements of database management systems. It introduces students to the concepts of logical and physical relationships in a data model and the concepts of inner and outer joins. Students will use a computer aided software engineering (CASE) tool to design, create, and query a database.
